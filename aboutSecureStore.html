Secure Store Vulnerable App

Secure Store is an intentional vulnerable Android application that is created by @srini0x00 to educate developers/pentetration testers/students and anyone who is interested in Mobile Security. Most of the common Mobile App Vulnerabilities have been covered. No clues have been provided on where the vulnerabilities are located. Go, find them and break them.

If you have any suggestions or feedback please feel free to contact us at @srini0x00

Some of the vulnerabilities covered

  • SQL Injection in REST APIs
  • Leaking sensitive data - REST APIs
  • Insecure Direct Object References
  • Weak authorization
  • Insecure local data storage
  • Broken Cryptography
  • SQL Injection at client side
  • WebView Injection
  • Insecure Direct Object References
  • Side Channel Data Leakage
  • Insufficient Transport Layer Security
  • Developer backdoors

Happy Hacking!
